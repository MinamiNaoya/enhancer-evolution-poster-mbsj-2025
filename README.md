# enhancer-evolution-poster-mbsj-2025
## 種横断エンハンサー予測AI：がん耐性・長寿動物に迫る新アプローチ

### 概要
1975年イギリスの疫学者Petoは自然界のあるパラドックスに気がついた。体が大きくなればなるほど体細胞数が多くなるため、複製の際のリスクなどが増加してがんにもなりやすくなるはず、しかし実際はそうではなく、体が大きいのにもかかわらず、がんになりにくいと報告されているゾウや体長18mの巨体をもちながら200年以上生きるホッキョククジラなどが存在する。このように哺乳類の中には極端にがんになりにくい動物や長寿な動物が存在する。一方で同じ1975年、アメリカの大学で研究を進めていたKingとWilsonは、チンパンジーとヒトの比較研究を通して、種の違いは遺伝子ではなく、遺伝子調節にあるという仮説を提唱した。そこから40年が立ちゲノム解析が活発になりVillarらは、遺伝子発現調節にかかわる非コード領域に存在するエンハンサーやプロモーターを進化的に比較し、エンハンサーが取得医的進化をしていることを示した。そこでわれわれは極端にがんになりにくい動物や長寿動物は、独自に進化した結果、がん耐性や長寿という形質を獲得しており、その根源にはエンハンサーの進化があるという仮説を立てた。現在、がん耐性・長寿動物のエンハンサー解析をすることは、そのサンプル入手の難しさや実験技術的な難しさから困難である。
### 目的
がん耐性動物・長寿動物のエンハンサー進化解析を行うための基盤となる種横断エンハンサー予測AI、EvoPhantの開発
### 方法
DNA言語モデルであるNucleotide Transformerをベースとし、複数のエンハンサーデータセットを用いてファインチューニングし、下流でLightGBMを用いて分類する。
