# enhancer-evolution-poster-mbsj-2025
## 種横断エンハンサー予測AI：がん耐性・長寿動物に迫る新アプローチ

### 概要
1975年、イギリスの疫学者 Peto は自然界に存在するある重要なパラドックスに気づいた。通常、体が大きくなるほど体細胞数が増加し、細胞分裂の回数や複製エラーの蓄積リスクも高まるため、がん発症率は上昇するはずである。しかし実際には、体が大きいにもかかわらず、がんになりにくい動物が存在する。たとえば、体重 6 トンを超えるゾウや、全長 18 m・寿命 200 年以上とされるホッキョククジラは、いずれも非常に低いがん発生率を示す。哺乳類の中には、このように極端ながん耐性や長寿を示す種が存在する。

同じく 1975 年、アメリカの大学で研究を進めていた King と Wilson は、チンパンジーとヒトの比較研究を通じて、「種間の違いは遺伝子配列そのものではなく、遺伝子調節の違いに由来する」という仮説を提唱した。その後 40 年の間にゲノム解析技術が飛躍的に進歩し、Villar らは、遺伝子発現調節に関わる非コード領域（エンハンサーやプロモーター）を多種哺乳類で比較することで、エンハンサーが種特異的に進化しやすい領域であることを明らかにした。

これらの知見を踏まえ、我々は「極端ながん耐性・長寿を示す動物では、独自に進化したエンハンサーが、がん耐性や長寿といった形質の獲得に寄与している」という仮説を立てた。しかし、これらの動物におけるエンハンサー解析は、サンプル入手の困難さおよび実験技術的な制約により、従来は容易ではなかった。
### 目的
がん耐性動物・長寿動物のエンハンサー進化解析を行うための基盤となる種横断エンハンサー予測AI、**EvoPhant**の開発

### 結果
ienhancerベンチマークを用いて、先行研究を上回った。特にエンハンサーの統合データベースEnhancerAtlasの豚の心臓データに対して高精度を示した。一方でハダカデバネズミのChIP-seq由来データに対しては他のベンチマークよりも性能が低下した。　　
ヒト・マウスのin vivo検証エンハンサーデータセットに対して種横断性の評価のためLOOSVを行った。
**EvoPhant**は種横断のエンハンサー予測可能性を示した。
